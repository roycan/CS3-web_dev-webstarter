üìö Instructions for High School Students

Method 1: Using Web-Safe Fonts (Easiest)

Step 1: Open your styles.css file in your text editor.

Step 2: Find the h1 section that looks like this:

h1 {
  font-weight: bold;
  font-size: 1.5rem;
  font-family: cursive;
}

Step 3: Replace cursive with any web-safe font:

h1 {
  font-weight: bold;
  font-size: 1.5rem;
  font-family: Arial, sans-serif;
}

Method 2: Using Google Fonts (More Options!)

Step 1: Add this line to the <head> section of your HTML file (before the closing </head> tag):

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

Step 2: Then use the font in your CSS:

h1 {
  font-weight: bold;
  font-size: 1.5rem;
  font-family: 'Roboto', sans-serif;
}

Want different Google Fonts? Visit fonts.google.com, pick a font, and copy the link they provide!

Example Changes:

Professional look with Arial:

h1 { font-family: Arial, sans-serif; }

Classic look with Times New Roman:

h1 { font-family: 'Times New Roman', serif; }

Modern look with Google Font (Montserrat):

h1 { font-family: 'Montserrat', sans-serif; }

Fun look with Google Font (Pacifico):

h1 { font-family: 'Pacifico', cursive; }

üí° Pro Tips:

  • Always include backup fonts: font-family: 'Roboto', Arial, sans-serif;
  • Web-safe fonts work everywhere without extra setup
  • Google Fonts give you more beautiful options but need the link in your HTML
  • Put font names with spaces in quotes: 'Times New Roman'
  • Test on different devices to make sure fonts look good everywhere
  • Don't use too many different fonts - 2-3 max for a clean look

üé® Font Style Recommendations:

  • Professional websites: Roboto, Open Sans, Lato, Arial
  • Creative projects: Pacifico, Dancing Script, Lobster
  • Modern & clean: Montserrat, Oswald, Source Sans Pro
  • Classic & elegant: Playfair Display, Merriweather, Georgia
  • Bold & strong: Impact, Oswald, Bangers
  • Fun & playful: Comic Sans MS, Righteous, Creepster
  • Tech/gaming: Orbitron, Press Start 2P, Courier New

üîß Advanced Tips:

Different fonts for different headers:

h1 { font-family: 'Montserrat', sans-serif; }
h2 { font-family: 'Open Sans', sans-serif; }
h3 { font-family: Arial, sans-serif; }

Change all text on your page:

body {
  font-family: 'Roboto', Arial, sans-serif;
}
